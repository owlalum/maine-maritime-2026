# Maine & Maritime 2026

A trip-planning app for a 2026 journey through Maine and the Maritime provinces of Canada (New Brunswick, Nova Scotia, Prince Edward Island, and Newfoundland & Labrador).

## Live site

Hosted via GitHub Pages: https://owlalum.github.io/maine-maritime-2026/

## Project goals

- Plan and visualize the route across Maine and Atlantic Canada
- Track stops, lodging, ferries, and points of interest
- Keep a shared itinerary that's accessible from any device

## Status

Early scaffold. The site currently serves a placeholder `index.html`; features will be built out from here.
